*Payin Crypto
1. Пользователь оформляет заявку
    1.1 Выбирает ID
    1.2 Выбирает валюту
    1.3 Вводит сумму
2. Запрос отправляется на шлюз криптоплатежей.
3. Пользователю отправляется сообщение, содержащее адрес кошелька (вернул шлюз) и инструкции.
4. Пользователь совершает платеж. Шлюз вызывает callback.
5. В обработчике callback отправляется уведомление админам с возможностью подтвердить оплату.
6. Админ подтверждает оплату. Пользователю отправляется уведомление.

*Payin special
1. Пользователь оформляет заявку
    1.1 Выбирает ID
    1.2 Выбирает валюту
    1.3 Вводит сумму
    1.4 Указывает реквизиты для вывода
2. Пользователю отправляется сообщение с реквизитами оплаты. Админам сообщение с возможностью отклонить или подтвердить заявку.
3. Админ подтверждает или отменяет заявку. Пользователю отправляется уведомление.

*Payout crypto
1. Пользователь оформляет заявку
    1.1 Выбирает ID
    1.2 Выбирает валюту
    1.3 Вводит сумму
    1.4 Указывает кошелек для вывода
2. Отправляется уведомление админам, с возможностью подтвердить или отклонить заявку.
3. Если админ отклоняет заявку - пользователю отправляется уведомление. Стоп.
4. Если админ подтверждает заявку, то отправляется запрос на шлюз.
5. Шлюз вызывает callback. Пользователю отправляется уведомление.

*Payout special
1. Пользователь оформляет заявку
    1.1 Выбирает ID
    1.2 Выбирает валюту
    1.3 Вводит сумму
    1.4 Указывает реквизиты для вывода
2. Отправляется уведомление админам, с возможностью подтвердить или отклонить заявку.
3. Админ подтверждает или отменяет заявку. Пользователю отправляется уведомление.
